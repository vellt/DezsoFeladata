# DEZSŐ FELADATA

Télapó a karácsonyi körútjának elemzését Dezsőre, a Pingvinre bízta. Azonban Dezsőnek még az összeadás is nehezen megy, így hát felkereste a Baross Gábor Technikum végzős Szoftverfejlesztő és -tesztelő diákjait, hogy segítsenek neki elemezni a télapótól kapott feladatokat C# nyelven, LINQ-lekérdezések segítségével.
